Ember-tagify is an addon that wraps [Tagify](https://github.com/yairEO/tagify) - tags input component.

## Compatibility
------------------------------------------------------------------------------

* Ember.js v3.16 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


## Installation
------------------------------------------------------------------------------

```
ember install ember-tagify
```
